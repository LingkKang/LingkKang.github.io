LAC

  • Language: a set of words
  • Word: a sequence, or string, of symbols
  • DFA: deterministic finite automata
    1. A finite set of states
    2. A finite set of input symbols, the alphabet
    3. A transition function
    4. An initial state (start state)
    5. A set of final states (accepting states)
  • Transition table: a table to represent finite automata
  • Transition diagram: a diagram to finite automata
  • NFA: nondeterministic finite automata
    1. A finite set of states
    2. A finite set of input symbols, the alphabet
    3. A transition function
    4. A set of initial states (start states)
    5. A set of final states (accepting states)
  • Extended transition function (delta hat): a function that takes a state and a string of input symbols and returns a state
  • Subset construction: a method to convert an NFA to its equivalent DFA
  • Regular expression: a string that describes a set of strings
    1. L(∅) = ∅
    2. L(ε) = {ε}
    3. L(x) = {x} where x ∈ Σ
    4. L(E + F) = L(E) ∪ L(F)
    5. L(EF) = L(E)L(F)
    6. L(E*) = L(E)*
    7. L((E)) = L(E)
  • Table-filling algorithm: a method to minimize a DFA
    • For a DFA (Q,Σ,δ,q0,F), p,q ∈ Q are equivalent states if and only if, for all w ∈Σ∗, ˆδ(p,w) ∈ F ⇔ ˆδ(q,w) ∈ F.
    • If two states are not equivalent, then they are distinguishable.
  • Regular language: a language that can be recognized by a computer with finite memory
  • Pumping lemma: a method to prove that a language is not regular
    • For any regular language L, there exists a number p (the pumping length) where, for any string s in L of length at least p, s can be divided into three pieces, s = xyz, such that
      1. |xy| ≤ p
      2. |y| > 0
      3. For all i ≥ 0, x(y^i)z ∈ L
  • CFG: context-free grammar
    1. A finite set of nonterminal symbols (nonterminals)
    2. A finite set of terminal symbols (terminals)
    3. A finite set of productions
    4. A distinguished start symbol that is nonterminal
    5. Nonterminals and terminals are disjoint
  • Linear grammar: a grammar in which each production has at most one nonterminal in its right-hand side
    • Left-linear: leftmost (first) symbol in the right-hand side of each production
    • Right-linear: rightmost (last) symbol in the right-hand side of each production
  • All regular languages are context-free.
  • Derivation tree: a tree that represents the process of deriving a string from the start symbol of a CFG
  • Ambiguity: a grammar is ambiguous if there exists a string that has more than one derivation tree, or more than one leftmost derivation, or more than one rightmost derivation
  • PDA: pushdown automata
    1. A finite set of states
    2. A finite set of input symbols, the alphabet
    3. A finite set of stack symbols
    4. A transition function
    5. An initial state
    6. An initial stack symbol
    7. A set of final states
  • DPDA: deterministic pushdown automata
    • iff |δ(q, x, z)| + |δ(q,ϵ,z)| ≤ 1
  • CFL: context-free language
    • Context-free languages can be described by context-free grammars and can be processed by a pushdown automata.
  • parsing:
    • top-down parsing: construct the parse tree from the root and process downward
    • bottom-up parsing: construct the parse tree from the leaves and process upward
  • Recursive-descent parsing: a way to implement top-down parsing
  • Full backtracking
  • predictive parsing
  • LL(1) grammar
  • Turing machine
    1. A finite set of states
    2. A finite set of symbols (the alphabet)
    3. A finite set of tape symbols such that the alphabet is a subset of the tape symbols
    4. A transition function
    5. An initial state
    6. A blank symbol which belongs to the tape symbols but not to the alphabet
    7. A set of final states
  • Decide: a Turing machine decides a language if it accepts it and it never enters an infinite loop
  • Grammar
